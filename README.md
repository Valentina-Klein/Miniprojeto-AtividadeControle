# Miniprojeto-AtividadeControle
Atividade Prática – Git e GitHub da matéria Gerência de Configuração e Controle de Versão

Este é um miniprojeto para praticar conexões com Git e GitHub. O projeto inclui um arquivo HTML com um layout simples de página introdutória.

Recursos da página: 
 - Foto de perfil
 - Nome e breve biografia
 - Lista de hobbies ou interesses
